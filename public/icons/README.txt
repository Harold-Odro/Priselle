PRISELLE — MASTER LOGO KIT
===========================

The core Priselle identity. Use this for the parent brand,
and use the sub-brand versions (Freight & Logistics, Sourcing & Trade)
only when context demands it.

TYPOGRAPHY
----------
  Display: Fraunces (or a refined serif like Caslon, Baskerville)
  Body:    Inter Tight (or a neutral geometric sans)
  
  The exported PNGs/JPGs use Liberation Serif Bold as a stand-in
  because Fraunces isn't embedded in the image rendering pipeline.
  When finalising print collateral, substitute Fraunces for the
  proper optical character.


USAGE QUICK-NOTES
-----------------
  - SVG everywhere possible — it's sharp at any size and tiny in file size
  - Transparent PNG when placing over photos, textures, or coloured surfaces
  - Cream/White JPGs for email signatures and systems without transparency
  - Minimum clear space around the mark = half the icon's width on all sides
  - Minimum legible size: 24px for the icon alone; 100px wide for the full lockup
  - Never stretch, rotate, recolour outside the palette, or add drop shadows
  - Never place the forest-on-cream mark on a dark background — use the
    forest inverted version for dark contexts
